Cloud Storage Configuration

The hams_s3 module adds a dedicated Cloud Storage block to Odoo's General Settings (Settings > General Settings).

Configuration Fields

  • Use Amazon S3 Storage: Enables the S3 backend. If the required OCA dependencies are missing, a warning alert will be displayed directing the administrator to run the setup script.
  • AWS Host: The endpoint URL for the S3-compatible service (leave blank for default AWS).
  • AWS Access Key: The Access Key ID for authentication.
  • AWS Secret Key: The Secret Access Key for authentication.
  • AWS Bucket: The name of the S3 bucket to store attachments.
  • AWS Region: The geographic region of the bucket (e.g., `us-east-1`).
