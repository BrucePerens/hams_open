SES Webhook Functionality

This module provides an interface to configure incoming email routing from Amazon SES via SNS webhooks.

UI Functionality

1. SES Webhook Domains

Path: SES Webhooks > Domains

  • List View: Displays configured domains, their associated company, and the Webhook URL.
  • Form View: Allows administrators to register a new domain and map it to a tenant company.
    • Secret Token: Automatically generated and used to authenticate requests.
    • Webhook URL: The URL to provide to Amazon SNS. It can be easily copied to the clipboard.
    • AWS Configuration Instructions: A dedicated tab providing step-by-step guidance on setting up the Amazon SNS Topic and Amazon SES Receipt Rule.

2. SES Webhook Logs

Path: SES Webhooks > Logs

  • List View: Shows a color-coded log of all received webhooks. Green for success, red for failed, and gray for ignored.
  • Form View: Displays detailed information about a specific webhook request, including:
    • Message ID, Date, Payload Type, and Status.
    • Raw Payload: The original JSON payload received from SNS.
    • Error Message: Displayed if the webhook failed to process.

Logs older than 30 days are automatically deleted by a scheduled action to preserve database space.
