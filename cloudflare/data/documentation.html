☁️ Cloudflare Edge Orchestration (cloudflare)

The Cloudflare Edge Orchestration module is your central command center for managing Cloudflare's CDN and Web Application Firewall (WAF). Fully optimized for Odoo 19, it automates caching, security, and connectivity between your Odoo server and the Cloudflare edge, significantly improving performance and security for one or many websites while maintaining strict Zero-Trust security standards.

1. Getting Started

To begin, you need to connect Odoo to your Cloudflare account. Navigate to Settings > Website and look for the Cloudflare Edge section. Configuration is website-specific, allowing you to manage multiple domains independently.

  • CF API Token: This is a secure token you create in your Cloudflare dashboard. It must have permissions for Zone.Cache Purge, Zone.Firewall Services, and optionally Account.Cloudflare Tunnel.
  • CF Zone ID: This unique ID identifies your domain in Cloudflare. You can find it on your domain's overview page in the Cloudflare dashboard.
  • CF Account ID: Your main Cloudflare account ID, required if you want to use Zero Trust Tunnels.

2. Making Your Site Fast: Cache Management

This module uses Cloudflare to store copies of your website content around the world, so it loads instantly for users everywhere.

  • Smart Caching: Odoo automatically tells Cloudflare how long to keep different types of content. Static files (like images and scripts) are cached for a year, while website pages are cached for 24 hours for public visitors.
  • Automatic Updates: When you edit a page, blog post, product, product review, or website menu, Odoo automatically tells Cloudflare to clear the old version from its cache. Page and post edits clear individual URLs, while menu changes trigger a full website cache refresh to ensure headers and footers are updated immediately. This all happens in the background, ensuring your visitors always see the latest content without slowing down your work.
  • Static File Sync: Every time your Odoo server starts, it checks if any of your theme or module files have changed. If they have, it triggers a global cache clear for those files across all your websites.
  • Manual Control: If you ever need to clear the cache manually, you can use the Purge Cache wizard under Website > Configuration > Cloudflare. You can clear everything, specific pages, or groups of content using tags.

3. Keeping Your Site Secure: Firewall & IP Bans

Protect your Odoo instance from bots and attackers using Cloudflare's security edge.

  • Honeypot Protection: The module integrates with silent "honeypot" traps. If a malicious bot triggers one of these traps, its IP address is instantly banned at the Cloudflare network level. You can review and manage these bans under Website > Configuration > Cloudflare > IP Bans.
  • Firewall Rules (WAF): You can manage advanced Cloudflare firewall rules directly from Odoo. Use the Pull action to import your existing rules or Push to deploy rules you've defined in Odoo. This is found under Website > Configuration > Cloudflare > WAF Rules.
  • Security Level: Quickly adjust how aggressive Cloudflare's security should be (e.g., enabling "Under Attack" mode if you are experiencing a surge of malicious traffic) using the Zone Settings Wizard.

4. Secure Connections: Zero Trust Tunnels

Cloudflare Tunnels allow you to host your Odoo server securely without opening any ports on your firewall. This creates a private, encrypted "tunnel" between your server and Cloudflare.

  • Easy Setup: Use the Generate Tunnel Command button in the settings. Odoo will create the tunnel in Cloudflare and give you a simple command to run on your server.
  • Management: View the health and status of your tunnels under Website > Configuration > Cloudflare > Tunnels. You can sync the list from Cloudflare or delete tunnels directly from the Odoo interface.

5. Advanced Integration & Security

For developers and advanced users, the module provides several tools designed for high security and performance:

  • Geographic Data: The module extracts location information (Country, City, etc.) provided by Cloudflare for every request. This data is handled safely even in background tasks where a standard web request might not be present.
  • Turnstile CAPTCHA: Securely verify Cloudflare Turnstile tokens on your public forms to prevent spam.
  • Safe Caching: Sensitive areas like your shopping cart, checkout, and portal pages are automatically excluded from edge caching to protect user privacy.

6. DNS & Custom Hostnames

  • DNS Records Management: View and manage your Cloudflare DNS records directly from the Odoo interface under Website > Configuration > Cloudflare > DNS Records.
  • Custom Hostname Provisioning: Allows multi-tenant setups where customers can bring their own domains. Odoo automatically provisions the Custom Hostname in Cloudflare and requests SSL certificates seamlessly.

7. Advanced Traffic & Security Controls

  • Analytics Dashboard: Access a comprehensive dashboard visualizing Cloudflare traffic, security events, and performance metrics right within Odoo.
  • Rate Limits & Cache Rules: Configure fine-grained rate limiting policies and custom cache rules to protect your application from abuse and optimize asset delivery.
  • Zero Trust Policies: Define and deploy Cloudflare Zero Trust policies to secure access to specific routes and internal applications.
  • Configuration Backups: Before deploying major changes from Odoo to Cloudflare, the system automatically creates pre-Odoo configuration backup workflows, ensuring you can safely revert if necessary.
