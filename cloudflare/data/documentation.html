☁️ Cloudflare Edge Orchestration (cloudflare)

The Cloudflare Edge Orchestration module acts as a command center for your Cloudflare CDN and Web Application Firewall (WAF). It automates caching, security, and connectivity between your Odoo server and Cloudflare's global network.

1. Configuration

To start using this module, you need to provide your Cloudflare credentials. Go to Settings > Website > Cloudflare Edge.

  • CF API Token: A token created in the Cloudflare dashboard with permissions for Zone.Cache Purge, Zone.Firewall Services, and optionally Account.Cloudflare Tunnel.
  • CF Zone ID: The unique ID for your website domain in Cloudflare.
  • CF Account ID: Your Cloudflare account ID (required if you want to use Tunnels).
  • Turnstile Secret: The secret key for Cloudflare Turnstile if you use it for CAPTCHA protection.

2. Cache Management

This module automatically handles caching to make your website faster.

  • Automatic Purging: When you update a website page, blog post, or product, the module tells Cloudflare to clear the old version from its cache so visitors see the latest content immediately.
  • Static Asset Sync: Every time your Odoo server restarts, the module checks if any static files (like images or CSS) have changed. If they have, it clears them from the global cache.
  • Manual Purge: You can manually clear the cache by going to Website > Configuration > Cloudflare > Purge Cache. You can choose to purge everything, specific web addresses (URLs), or specific "Cache-Tags".

3. Security & WAF

Protect your site using the Cloudflare Web Application Firewall.

  • IP Banning: If a malicious user is detected, their IP can be banned. You can view and manage these bans under Website > Configuration > Cloudflare > IP Bans.
  • Firewall Rules: You can manage advanced firewall rules directly from Odoo. Go to Website > Configuration > Cloudflare > WAF Rules. You can "Pull" existing rules from Cloudflare or "Push" your local Odoo rules to Cloudflare.
  • Security Level: You can quickly adjust your site's security level (e.g., to "Under Attack" mode) using the Zone Settings wizard in the Cloudflare menu.

4. Cloudflare Tunnels

Cloudflare Tunnels allow you to securely connect your Odoo server to the internet without opening any ports on your firewall.

  • Creating a Tunnel: In the Cloudflare settings, click Generate Tunnel Command. This will create a new tunnel in Cloudflare and give you a command to run on your server.
  • Managing Tunnels: View and manage your active tunnels under Website > Configuration > Cloudflare > Tunnels. Use the Sync Tunnels button to update the list from Cloudflare.

5. Developer & API Info

For developers, the module provides several anchors for integration:

  • [@ANCHOR: cf_execute_ban]: Logic for executing an IP ban.
  • [@ANCHOR: cf_action_lift_ban]: Logic for lifting an IP ban.
  • [@ANCHOR: enqueue_urls_base_url]: Logic for resolving URLs before purging.
  • [@ANCHOR: cf_tunnel_setup]: Logic for tunnel provisioning.
  • [@ANCHOR: ir_http_post_dispatch_headers]: Header injection logic.
