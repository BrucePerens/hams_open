Daemon Key Manager

The Daemon Key Manager module provides a secure way to manage Odoo API keys for external background daemons. It automates key generation, rotation, and distribution to secure environment files on the local filesystem.

Core Features

  • Secure Key Generation: Generates native Odoo API keys for service accounts.
  • Automated Rotation: Keys are automatically rotated every 60 days.
  • Secure File Export: Keys are exported to .env files with strict 0600 permissions.
  • Zero-Sudo Compliance: All operations are performed with minimum privilege using micro-service accounts.

API Usage

Other modules can register their daemons using the register_daemon API:

env['daemon.key.registry'].register_daemon(
    daemon_name="My Daemon",
    user_xml_id="my_module.my_service_account",
    env_file_path="/var/lib/odoo/daemon_keys/my_daemon.env"
)

Security Configuration

Environment files MUST be stored in /var/lib/odoo/daemon_keys/. This path is enforced by security constraints.

Service accounts used for daemons should have minimal privileges required for their tasks.

Troubleshooting

If a daemon fails with an AccessError, it should reload its environment file to pick up the newly rotated key.
