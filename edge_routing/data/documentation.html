Edge Routing Management

User & Administrator Configuration

  • Custom Domain Mapping: Administrators can map user vanity slugs to custom external domains. This allows users to host their profiles or user websites under their own root domain instead of a subpath.
  • Vanity URL Management: End-users can claim unique vanity slugs for their profiles, preventing collisions with system reserved routes.
  • Virtual Slug Fallback: If a vanity slug is not set or removed, the routing engine automatically falls back to resolving profiles via the operator's Callsign (login ID).

Integration Details

All custom domains and routes mapped within the UI are asynchronously synchronized to the external PagerDuty routing layer to ensure high availability and global load balancing.
